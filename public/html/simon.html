Captain Says!

Instructions

The objective is to match the order of the Captain's Buttons using the Player's Buttons. Each round the Captain's Buttons will light up, adding one more step each round. If you match the order your level will increase, but mess up and the game will be over.

Press the "Start" button to begin the game.

Click the buttons in the proper order then submit your attempt using the "Submit" button

To begin the next round simply press the "Next" button.
